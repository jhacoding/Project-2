This project predicts product's price using Machine Learning. The complete ML workflow has been implemented, starting from data cleaning to hyperparameter tuning and final model evaluation.

The objective is to build an accurate predictive model while demonstrating a complete Data Science pipeline.

Technologies Used :
a) Python
b) NumPy
c) Pandas
d) Matplotlib
e) Seaborn
f) Scikit-Learn

Exploratory Data Analysis (EDA)

Performed:

Missing value analysis
Duplicate value removal
Outlier detection
Feature distributions
Correlation analysis
Data visualization
Target variable analysis

Data Preprocessing

Implemented:

Data Cleaning
Handling Missing Values
Feature Engineering
Feature Extraction
Encoding Categorical Variables
Feature Scaling
Train-Test Split

Hyperparameter Tuning

Optimized models using:

GridSearchCV
RandomizedSearchCV
